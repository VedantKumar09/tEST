==================================================
7) TROUBLESHOOTING DURING RECORDING
==================================================
Issue: npm or script policy errors
Fix:
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

Issue: Port 8000 blocked
Fix:
Get-NetTCPConnection -State Listen | Where-Object { $_.LocalPort -eq 8000 }
Stop-Process -Id <PID> -Force

Issue: Frontend not loading on 5173
Fix:
cd C:\Users\Victus\Downloads\MindMesh\tEST\frontend
node node_modules/vite/bin/vite.js

Issue: OpenAI quota error during question generation
What to say:
"OpenAI quota is exhausted in this environment; local fallback generation is used to maintain continuity."

Issue: Phone camera not connecting
Fix:
1. Ensure same Wi-Fi.
2. Re-open QR link.
3. Accept HTTPS warning on phone.
4. Retry Start Camera on phone page.
